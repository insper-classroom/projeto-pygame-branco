Instruções

1. Pressione a tecla "W" no teclado para iniciar o jogo.

2. Seu objetivo é acertar o máximo de alvos possível em 20 segundos.

3. Use o mouse para mirar. Mova o cursor do mouse para apontar para o alvo desejado.

4. Clique com o botão esquerdo do mouse para atirar. Certifique-se de que sua mira esteja precisa antes de disparar.

5. Alvos aparecerão aleatoriamente na tela. Eles podem se mover ou ficar parados.

6. Acerte os alvos para ganhar pontos. Cada alvo acertado dará pontos.

7. Cuidado com as menininhas! Se você acertar uma menininha, perderá 1 ponto. Evite atingi-las a todo custo.

8. Na tela do jogo, haverá um cronômetro regressivo de 20 segundos. Você precisa acertar alvos dentro desse tempo.

9. O jogo terminará automaticamente quando o cronômetro chegar a zero.

10. Após o término do jogo, sua pontuação final será exibida na tela. Tente superar sua pontuação anterior!

11. Para jogar novamente, pressione a tecla "W" para reiniciar o jogo.
